# Release Notes: v0.9.0 Prerelease

## Features

- Static Cloudflare Workers-ready map application.
- Indonesia ADM2 polygon rendering without external basemap tiles.
- Search, province filter, region dropdown, click selection, and fit-to-Indonesia.
- Manual region coloring with multiple colors.
- Highlight list, remove, undo, reset.
- CSV import with local-only processing, preview, invalid-row reporting, duplicate detection, and ambiguous-match protection.
- Project JSON save/open and browser autosave.
- Legend editing.
- SVG and PNG export.
- Privacy and attribution documentation.

## Data

- Recommended source: geoBoundaries IDN ADM2 / HDX COD-AB Indonesia lineage.
- License: CC BY-IGO / CC BY 3.0 IGO as reported by source metadata.
- Boundary year represented: 2020.
- Production feature count: 519.
- Metadata reference rows inspected: 522.
- Unique metadata matches: 466.
- Ambiguous geometry names: 53.

## Prerelease reason

This is marked as `v0.9.0` rather than `v1.0.0` because live Cloudflare Workers deployment, cross-browser manual testing, and full topology validation were not completed in this environment.
